# Dev Browser

A lightweight web browser with built-in developer tools — built in Python. Browse the web and run JavaScript, all in one simple app.

## About

Dev Browser is a Python-based desktop web browser designed with developers in mind. It includes a simple set of dev tools that let you interact with pages directly — great for testing and tinkering.

## Features

- 🌐 Basic web browsing — navigate to any URL
- 🛠️ Built-in developer tools
- ⚙️ Run JavaScript directly in the browser
- 🐍 Built with Python — lightweight and hackable
- 💻 Available as both a `.py` script and a standalone `.exe`

## Getting Started

### Option 1 — Run the Executable

Download the latest release from the [Releases](https://github.com/Big-A-Software/Dev-browser/releases) page and run the `.exe` — no Python required.

### Option 2 — Run from Source

**Prerequisites:**
- [Python 3](https://www.python.org/downloads/)

```bash
git clone https://github.com/Big-A-Software/Dev-browser.git
cd Dev-browser
python dev_browser.py
```

> If any dependencies are missing, install them with:
> ```bash
> pip install -r requirements.txt
> ```

## Usage

1. Launch the app
2. Type a URL into the address bar and hit Enter
3. Use the built-in dev tools panel to inspect or run JavaScript on the page
